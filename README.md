# React + TypeScript + Vite

# React Question Cards

Простое и красивое приложение для изучения вопросов и ответов по React.  
В разработке как пет-проект.

## Технологии

- **React 19** + **TypeScript**
- **Vite** — быстрый и современный сборщик
- **React Router v7** — навигация по страницам
- **react-toastify** — красивые уведомления (тосты)
- **json-server** — фейковый REST API для хранения вопросов (локально)

## Установка и запуск

1. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/AbramenkoEkaterina/react-app-course
   cd react-app-course

Установите зависимости:Bash npm install
Запустите сервер с данными (json-server):npm run serverСервер запустится на http://localhost:8801
В отдельном терминале запустите приложение: npm run dev Приложение откроется по адресу http://localhost:5173 



npm run dev — запуск фронтенда в режиме разработки
npm run server — запуск json-server (данные хранятся в db.json)
npm run build — сборка проекта для продакшена
npm run lint — проверка кода ESLint + Prettier
npm run preview — локальный просмотр собранной версии
